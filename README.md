# Forgeborn

Forgeborn is a 2D atmospheric descent game built in **Godot 4** using **GDScript**.

You play as a small android descending into a vast, dark abyss left behind by the Forgebornâ€”an ancient civilization that replaced itself with machines in pursuit of immortality. The game focuses on mood, scale, and restraint rather than action or realism.

## Design Focus
- Tiny, pixel-perfect player sprite that remains readable while feeling small on screen
- Very dark environments with visibility centered around the player
- Consistent, restrained color palettes with rare biome-based exceptions
- Ambient sound design and silence as core storytelling tools
- Minimal UI (Health + Charge)
- Player decisions tracked for narrative consequences
